# Project Title

CHATAPP

## Description

This project was created using HTML, CSS, JavaScript, NodeJS & Socket.io. This chat application is sending and receiving messages in real-time using socket.io library.

## Getting Started

### Dependencies

- express: 4.18.2 version
- socket.io: version 4.5.4

### Installing

- Install/run Node.js and npm

### Executing program

- Open VS code terminal.
- Run npm install & start.

```
npm install
```
```
npm start
```

- Open two windows or Incognito.
- Enter localhost:5000 for each

## Help

If live-server crashed run ctrl C

```
^C
```
then run npm start in VS code terminal again.
```
npm start
```
